EqualityFilter

class EqualityFilter implements FilterInterface

EqualityFilter class.

The EqualityFilter can be used to compare a resource's field or metadata against a specific value.

EqualityFilters are one of the very basic filters and are typically used ensure the type of a document or to match entries of a content type:

{
  "equals": [{"doc": "sys.environment.sys.id"}, "staging"]
}
